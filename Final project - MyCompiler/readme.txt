Supported C subset:

        1. program "void main(){}"
        2. integer data type
        3. float data type
        4. declarations
        5. arithmetic expression
        6. if_then if_else
        7. Comparison expression
        8. printf function
        9. for loop
        10. while loop
        11. comment /**/
        12. ++ -- += -=


Note:
    printf function only accept one argument.


Compile:
    just type make in command line, and we will get three .j files (test1.j  test2.j  test3.j )


Execution:

    java -jar jasmin.jar test1.j
    java myResult

    java -jar jasmin.jar test2.j
    java myResult

    java -jar jasmin.jar test3.j
    java myResult

    OS: Ubuntu 18.04.4 LTS
    Processor: Intel® Core™ i5-6200U CPU @ 2.30GHz × 4
